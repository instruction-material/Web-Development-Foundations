# Web Development Foundations

Starter and solution material for the Web Development Foundations course.

## Labs

- `WDF1-Portfolio-Custom-Domain`
- `WDF2-Notes-App-with-MongoDB`
- `WDF3-Booking-Contact-App`
- `WDF4-Realtime-Chat-App`
- `WDF5-Separate-Deployment-Lab`

Each lab contains a `starter/` directory for students and a `solution/`
directory for instructor review.
